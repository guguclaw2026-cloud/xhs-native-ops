# Hook
小团队最该先交给 AI 的，不是创意

# Angle
先交给 AI 的，应该是重复又稳定的动作。

# Card Outline
- 别先让 AI 代替判断｜先让它接手重复动作，回报更稳定
- 第一步交给 AI 的活｜整理信息、抽重点、改格式、写初稿
- 不要一开始就交给 AI 的活｜价值判断、拍板决策、最后对外表达
- 为什么很多团队体感差｜不是 AI 没用，而是交错了活
- 真正轻下来的顺序｜重复动作 → 半结构化工作 → 最后才是复杂表达
- 一个简单判断标准｜离开你还能稳定完成的事，优先让 AI 接手
- 如果今天就要试｜先挑每周最烦但最标准化的一件事
- 你该得到的结果｜不是更忙，而是把脑子留给更值得你做的判断

# CTA
收藏这套顺序，等你准备开始分工时直接照着试。

# Notes
默认走 review-first。先把包做完整，再决定是否进入真实发布流程。
